# Blog Application (CRUD)
 
A simple Blog Application built using Node.js, Express.js, and EJS.
This project demonstrates basic CRUD operations (Create, Read, Update, Delete).

# Features

➕ Add new blog post
📖 View all blog posts
✏️ Edit existing post
❌ Delete post

# Technologies Used
Node.js
Express.js
EJS (Embedded JavaScript Templates)

# How It Works
Data is stored in an in-memory array (temporary storage)
Express handles routing
EJS is used to render dynamic HTML pages
